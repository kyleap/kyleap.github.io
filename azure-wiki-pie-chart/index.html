用 Mermaid 在 Azure wiki 繪製圖表 - 圓餅圖

2023-03-27 · 1 min · Kyle

在一條龍的 Azure DevOps 平台，當然也存在 KM 功能， 透過 Azure Wiki 與 Workitem 結合，撰寫全貌的文件，能協助團隊更有效地進行開發與協作，其中Wiki 支援 Mermaid 語法來繪製圖表，本篇以圓餅圖作為示範

透過 ::: mermaid 與 ::: 裡面放置 mermaid 語法，即可呈現圖表

::: mermaid

pie
    title 年度銷售量
    "Q1" : 2000
    "Q2" : 1500
    "Q3" : 3500
    "Q4" : 4000

:::

會產生以下結果

也可以加入 pie showData 讓 Label 呈現數值

也支援小數點數值

只可惜目前在 Azure wiki 無法針對樣式去做設定，但會跟著使用者的 theme 進行適當的配色
